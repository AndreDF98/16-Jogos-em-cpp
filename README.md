# 16 Jogos em C++

Implementação de 16 jogos clássicos usando C++, baseado na playlist [Let's make 16 games in C++/SFML!
](https://www.youtube.com/playlist?list=PLB_ibvUSN7mzUffhiay5g5GUHyJRO4DYr) de [FamTrinli](https://www.youtube.com/channel/UCC7qpnId5RIQruKDJOt2exw).
FamTrinli usou a biblioteca [SFML](https://www.sfml-dev.org/) para criar seus jogos, mas aqui implementei os mesmos jogos usando a biblioteca [SDL](https://www.libsdl.org/).
